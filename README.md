🎨 Miho Takahashi Studio - Creative Shop Portfolio
「日常に、もっと色彩を。」
デザインとフロントエンド技術を融合させ、色彩豊かな世界観を形にしたWebアプリケーション・ポートフォリオです。

📖 プロジェクト概要
このプロジェクトは、Webデザイナーおよびフロントエンドエンジニアとしてのスキルを証明するために制作されました。
ユーザーが作品の世界観に没入できる「Look Book」から、直感的な操作で買い物を楽しめる「Shop」機能まで、デザイン性と機能性の両立を追求しています。

🛠️ 技術スタック
制作時期や目的に合わせ、2つのアプローチで実装を行っています。

1. バニラJS & Alpine.js 版 (軽量・高速プロトタイプ)
Framework: HTML5 / Tailwind CSS

Interaction: Alpine.js (状態管理・カート機能)

Icons: Lucide Icons

Fonts: Playfair Display (Serif)

2. React (Next.js) 版 (拡張性・コンポーネント指向)
Framework: React / TypeScript

Animation: Framer Motion (motion/react)

Styling: Tailwind CSS

Icons: Lucide React

✨ 注力したポイント
🌈 視覚体験とブランディング
シチュエーションに応じたナビゲーション: スクロール量に応じて透過からホワイト背景（ブラー効果付き）へ変化するナビゲーションバーを実装。常に操作性を損なわない設計にしました。

洗練されたタイポグラフィ: 高級感のある Playfair Display を見出しに採用し、ブランドの信頼感と芸術性を表現しています。

🛒 インタラクティブなカート機能
シームレスなUI: ページ遷移をせず、サイドパネル形式で開閉するショッピングカート。

リアルタイム更新: 商品の追加、数量変更、削除、合計金額の計算を状態管理（Alpine.js / React Hooks）によって瞬時に反映。

マイクロインタラクション: ボタンのホバーアニメーションや、カートバッジの通知演出など、ユーザーに「心地よさ」を与える演出を加えています。

📱 モバイルフレンドリー
レスポンシブデザインを徹底し、スマートフォンでもPCと同様のブランド体験ができるよう、グリッドレイアウトやタッチターゲットを最適化しています。

📂 構成 (Sections)
Hero: インパクトのあるキャッチコピーとアクセントカラーによる導入。

Concept: ブランドの想いを伝えるメッセージ。

Look Book: 季節ごとのコレクションを視覚的に訴求。

Shop: 商品一覧、詳細モーダル、カート追加機能。

Contact: フッター、SNS連携、ニュースレター登録。
